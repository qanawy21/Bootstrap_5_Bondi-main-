# Bondi – Bootstrap 5 Landing Page

Bondi is a modern, clean, and fully responsive landing page website built using the powerful **Bootstrap 5** framework. Designed to professionally showcase creative agencies, portfolios, or corporate businesses, this template utilizes Bootstrap's flexible grid system and pre-built components alongside custom CSS to deliver a polished, high-quality user experience across all devices.

---

## 🌐 Demo (Live Preview)

**👉 Live Demo:** [https://qanawy21.github.io/Bootstrap_5_Bondi-main-/](https://qanawy21.github.io/Bootstrap_5_Bondi-main-/)

---

## 💻 Used Technologies

* **HTML5** (Semantic markup)
* **CSS3** (Custom styling and overriding framework variables)
* **Bootstrap 5** (Grid system, responsive utilities, and UI components)
* **JavaScript** (Bootstrap Bundle for component interactivity like Navbars)
* **Font Awesome** (Icons)
* **Google Fonts**
* **GitHub Pages** (Hosting)

---

## ✨ Key Features

* Built entirely on the robust Bootstrap 5 framework
* Fully responsive design optimized for mobile, tablet, and desktop viewing
* Clean and modern User Interface (UI) suitable for agencies and startups
* Interactive responsive Navbar with a mobile toggle menu
* Well-structured sections including Hero, Features, Portfolio/Work, and Team
* Smooth integrations of Font Awesome icons for visual enhancement
* Clean, readable, and easily customizable source code

## 📥 Installation Instructions for Local Setup

1. **Clone the repository:**

```bash
git clone https://github.com/qanawy21/Bootstrap_5_Bondi-main-.git

```

2. **Navigate to the project directory:**

```bash
cd Bootstrap_5_Bondi-main-

```

3. **Run the project:**

open `index.html` in your browser

> No additional build tools are required—just open the file directly to view the site!
